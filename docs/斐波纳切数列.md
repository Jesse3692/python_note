# 斐波纳切数列

``` Python
# 斐波纳切数列（生成器版）
def fibon(n):
    a = b = 1
    for i in range(n):
        yield a
        a, b = b , a + b
for x in fibon(10):
    print(x)
```

``` Python
# 普通版
def fibon(n):
    a = b= 1
    result = []
    for i in range(n):
        result.append(a)
        a, b = b, a + b
    return result
fibon(10)
```
