# mongodb

使用mongodb shell连接到数据库

```shell
C:\Users\Administrator\AppData\Local\Programs\mongosh\mongosh.exe "mongodb://localhost:27017" --username root --authenticationDatabase admin
```

查询所有数据库

```shell
show collections;
```

切换/创建数据库

```shell
use yourdb;
```

查询所有集合

```shell
show collections;
```
