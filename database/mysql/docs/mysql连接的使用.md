# MYSQL连接的使用

在MySQL中可以使用JOIN来进行多表查询数据

JOIN按照功能大致可分为三类：

- INNER JOIN（内连接或等值连接）：获取两个表中字段匹配关系的记录
- LEFT JOIN（左连接）：获取左表所有记录，即使右表没有对应匹配的记录
- RIGHT JOIN（右连接）：与LEFT JOIN相反，用于获取右表所有记录，即使左表没有对应匹配的记录
